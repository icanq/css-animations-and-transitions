Animasi Menggunakan JavaScript

Pada intinya, Animasi JavaScript itu tentang mengubah HTML atau CSS element/property pada setiap waktu/ waktu tertentu, bisa berupa pergerakan sebuah elemen, bisa juga berupa ngatur elemennya ngilang dulu terus balik lagi, yang jelas animasi pada JavaScript itu mengubahnya di bagian file JavaScriptnya atau dalam yang ada di file html kamu. Kenapa pake JavaScript? karena kamu bisa lebih leluasa dalam mengatur setiap elemen yang ada pada web yang kamu buat melalui yang namanya DOM (Document Object Model) dan kamu bisa set animasinya sesuai dengan ketentuan tertentu sehingga bisa jadi lebih dinamis.
